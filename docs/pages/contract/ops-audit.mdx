# C0 operations audit — agent-api-spec §8–9 vs C# `Models/`

Audited 2026-07-19 against `blades-in-the-sheets/Models/` (authoritative).
Verdicts: **adopted** (catalog matches source), **corrected** (catalog wrong or
incomplete; contract follows source or spec §7.2), **diverged** (contract
deliberately differs from source; noted), **dropped**.

## Error codes (contract enumeration)

`NOT_FOUND, VALIDATION, STALE_REVISION, RETIRED, CONFIRM_REQUIRED, DUPLICATE,
SLOT_FULL_FATAL, CANNOT_HEAL, ARMOR_NOT_AVAILABLE, ABILITY_MAXED,
CANNOT_LEVEL_UP, RATING_MAXED, UPGRADE_MAXED, INSUFFICIENT_FUNDS, SATCHEL_FULL,
OVER_BULK, NO_COMMITMENT, COMMITMENT_LOCKED, NO_HISTORY, GAME_NOT_FOUND,
PAYLOAD_TOO_LARGE`.

Catalog codes dropped: `STRESS_OVERFLOW` (not an error), `HARM_SLOT_FULL`
(replaced by spillover + `SLOT_FULL_FATAL`), `INVALID_INTENSITY`/`FRIEND_EXISTS`
(→ `VALIDATION`/`DUPLICATE`).
