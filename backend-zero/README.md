# backend-zero

Track Z — Zero ([zerolang.ai](https://zerolang.ai)) backend for paperclips-in-the-dark.

## Toolchain install (Z0)

```sh
curl -fsSL https://zerolang.ai/install.sh | bash
export PATH="$HOME/.zero/bin:$PATH"
zero --version
# zero 0.3.4 (build 5b3a90a) — version pinned by the spike memo
```

- Release assets: <https://github.com/vercel-labs/zerolang/releases>
- Repo: <https://github.com/vercel-labs/zerolang>
